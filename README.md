# Payment Gateway Project

A full-stack payment gateway simulation inspired by Razorpay and Stripe.

## Features
- Merchant onboarding with API key authentication
- Order creation and payment processing
- UPI and Card payment simulation
- Hosted checkout page
- Merchant dashboard
- Dockerized deployment

## Tech Stack
- Backend: Node.js / Java Spring Boot (TBD)
- Frontend: React
- Database: PostgreSQL
- Containerization: Docker & Docker Compose

## How to Run
```bash
docker-compose up -d
